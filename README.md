## Задание на стажировку IOS Авито(Dostavito).

Этот проект представляет собой iOS-приложение для поиска и просмотра товаров, с использованием API для загрузки данных. Приложение состоит из трех основных экранов:

- Экран поиска товаров с постраничной загрузкой и фильтрами.
- Экран детальной информации о товаре с возможностью добавить в «Список покупок».
- Экран списка покупок с возможностью делиться и синхронизации с карточками товаров.

# Установка проекта:

1. [Клонируйте](https://github.com/Ussehh/AvitoInternshipWinter2025) репозиторий
2. Откройте проект в Xcode: Откройте файл ios-internship-winter-2025.xcodeproj в Xcode.
3. Настройте API: Для работы с API необходимо указать ключ в файле Network(Folder) -> NetworkManager.swift:
-       static let baseURL = "вставьте_свой_API_ключ_сюда"
4. Запуск на симуляторе или устройстве: Выберите симулятор или подключённое устройство и нажмите Cmd + R, чтобы запустить проект.

## Реализованные функции 

### Экран поиска
- Строка поиска для ввода запроса
- История поиска: приложение сохраняет до 5 последних текстовых поисковых запросов.
- Поиск по тексту: приложение поддерживает кастоную выдачу ячеек, в зависимости от категории(два товара в ряду, один товар в ряду).
- Пагинация: подгрузка товаров постранично по мере прокрутки.
- Фильтры: возможность фильтровать товары прямо на экране поиска, с использованием Bottom Sheet. Фильтры поддерживают комбинирование всех типов, включая текстовые (поисковые) запросы. 
- Empty-state: при отсутствии товаров или ошибке загрузки отображается сообщение об отсутствии товаров с возможностью перезапроса через жест свайпа.
- Сохранение настроек поиска: сохраняются последние поисковые запросы и выбранные фильтры.

### Экран детальной информации о товаре

- Детальная информация о товаре: изображение, заголовок, описание, цена, категория.
- Отображение ошибки загрузки: если изображение не загрузилось, показывается изображение - placeholder.
- Поделиться товаром: возможность поделиться информацией о товаре через системные средства.
- Возврат к экрану поиска: возможность вернуться назад на экран поиска товаров.
- Добавление в «Список покупок»: возможность добавить товар в список покупок. Если товар уже в списке, при нажатии происходит переход к списку покупок.
- Галерея изображений: просмотр всех доступных изображений товара в виде галереи + полноэкранный режим.
- Поддержка ScrollView: приложение адаптируется к размерам экрана, автоматически растягивая содержимое, если элементы не помещаются.

### Экран списка покупок
- Персистентность списка: товары, добавленные в список покупок, сохраняются между запусками приложения.
- Товары в списке: отображаются изображения товаров, цена, количество и возможность изменить количество или удалить товар. Eсли изображение не загрузилось, показывается изображение - placeholder
- Поделиться списком: возможность отправить список покупок в виде текста в мессенджере или сохранить в «Заметки».
- Детальная карточка: возможность перехода к подробной информации о товаре из ячейки в списке покупок.
- Синхронизация с карточками товара: изменения в списке покупок синхронизируются с экранами поиска и карточки товара.

#### Каждый экран приложения поддерживает три состояния:

- Отображение контента — когда данные успешно загружены и отображаются пользователю.
- Состояние загрузки — отображается индикатор загрузки во время получения данных.
- Отображение ошибки — показывается сообщение об ошибке, если возникла проблема с загрузкой данных.
  
### Задействованный стек

- Язык программирования: Swift.
- UI: UIKit.
- Сетевые запросы: URLSession для взаимодействия с API.
- Поддержка iOS: 13.0 и выше.
- Кеширование: NSCache
- Память: UserDefaults
- Работа с потоками: GCD
- Приложение не зависит от сторонних библиотек и фреймворков; весь функционал реализован с использованием стандартных фреймворков iOS.
